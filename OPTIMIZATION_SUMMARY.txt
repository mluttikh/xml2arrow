║                         KEY OPTIMIZATIONS                                 ║
╚══════════════════════════════════════════════════════════════════════════╝

 1. UTF-8 FAST PATH
    └─ Eliminated String allocation on every text node
    └─ Impact: Zero-cost for valid UTF-8 (99% of XML)
    
 2. ATTRIBUTE BUFFER REUSE  
    └─ Eliminated per-attribute String allocations
    └─ Impact: Critical for attribute-heavy documents
    
 3. INCREASED BUFFER CAPACITIES
    └─ XML buffer: 256 → 4096 bytes (16x)
    └─ Field buffer: 32 → 128 bytes (4x)
    └─ Impact: Reduced reallocation overhead
